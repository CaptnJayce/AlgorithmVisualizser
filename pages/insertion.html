insertion sort

explanation

how does it work:

> starts from the second element in the input list
> if element before it is greater then swap
> keep until an element less than it is met
> repeat until sorted

use cases:

> when a data set is nearly sorted
> if there are memory constraints
> as a teaching tool

code:

function insertionSort(array, n)  
{  
    let i, key, j;  
    for (i = 1; i < n; i++) 
    {
